# uma-ev

スクレイピングの際には、アクセス頻度とデータ利用目的に細心の注意を払い、対象サイトの利用規約に注意してください。本ツールの利用により、対象サイトから通信制限、アカウント停止、または損害賠償請求等の措置が取られた場合でも一切の責任を負いません。
