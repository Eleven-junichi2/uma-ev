# 設計

## データのディレクトリ構造

- `data/`: データのルートディレクトリ
- `data/`: : データ分析・予想に用いるカスタムデータを置く 
- `data/race`: 特定レースのデータを格納する
- `data/race/年4桁-月2桁-日2桁/競馬場/レース番号`: この形式でレース別に格納
- `data/race/年4桁-月2桁-日2桁/競馬場/レース番号/factors`: そのレースでのデータ分析・予想に用いるカスタムデータを置く 
- `data/netkeiba_id_map/`: netkeibaのデータ上のidと、ある種類の値との対応を記載するマップファイルを格納
- `data/running`: 走法・走行データを格納
- `data/running/horses`: 各馬ごとのファイル別で走法・走行データを格納
- `pedigree.csv`: 5代血統表を各馬行ごとに記載
